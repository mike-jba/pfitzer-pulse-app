# Pfitzer Pulse

## What This Is

Pfitzer Pulse is an internal call intelligence dashboard for Pfitzer Pest Control. It automatically processes phone call recordings from Voice for Pest (portal.atscall.me), transcribes them via Deepgram, analyzes them with Claude (Anthropic), and displays call insights, quality audits, and daily recaps for Karen Kieffer and Garret Pfitzer.

## Core Value

Karen and Garret can understand what's happening on every call without listening to recordings — categories, sentiment, flags, quality scores, and daily summaries in one place.

- **Auth deferred:** No Supabase Auth in v2; settings must work without user sessions

- **Vercel deployment:** No GitHub auto-deploy; manual `vercel --prod` to redeploy
